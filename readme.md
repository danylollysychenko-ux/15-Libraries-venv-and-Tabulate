# Overview of Libraries Venv and Tabulate

## Pip
- pip is the standard packet manager for python
- used to istall and manage libraries and dependencies from the Python Package Index

## Venv - Sets us a virtual environment 
- a fresh copy of the latest editon of your python
- when you install and activate a virtual environment, all libararies installed with pip are local to that project
- avoids version conflicts
- generally a best practice

## Tabulate
- a lightweight tool for pretty-printing tabular data in various formats. It is commonly used to transform raw data structures like lists or dictionaries into readable tables for terminal output, Markdown, or HTML.
